
# 📚 Quiz: Data Processing & Interfaces

An interactive quiz application to test your knowledge about data processing, computer fundamentals, and IT interfaces.

## 🌟 Features

✅ 20 Multiple-Choice Questions  
✅ Instant Feedback (Correct/Incorrect)  
✅ Explanations after each question  
✅ Color-coded feedback (Green/Red)  
✅ Progress indicator  
✅ Automatic scoring & grades  
✅ Responsive design (all devices)  
✅ No external dependencies  
✅ Offline compatible  

## 🎯 Topics Covered

- Data vs. Information vs. Messages
- EVA Principle (Input → Processing → Output)
- CPU, RAM, and Storage
- Hardware vs. Software
- Input/Output Devices
- Analog vs. Digital Signals
- Signal Transmission & Disturbances
- Information Channels
- USB 3.2, Hot-Plug, Thunderbolt 4
- Device Drivers
- RS-232 Serial Communication

## 🚀 Quick Start

### Option 1: Direct Browser
1. Download all files
2. Double-click `index.html`
3. Start the quiz!

### Option 2: Local Server
```bash
# Python 3
python3 -m http.server 8000

# Then open: http://localhost:8000
```
